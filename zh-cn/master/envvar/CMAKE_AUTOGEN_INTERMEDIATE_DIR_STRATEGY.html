CMAKE_AUTOGEN_INTERMEDIATE_DIR_STRATEGY¶

Added in version 4.2.

This is a CMake Environment Variable. Its initial value is taken from the calling process environment.

CMAKE_AUTOGEN_INTERMEDIATE_DIR_STRATEGY is a string specifying the strategy to use for autogen-related target intermediate directories. It initializes the CMAKE_AUTOGEN_INTERMEDIATE_DIR_STRATEGY variable.
