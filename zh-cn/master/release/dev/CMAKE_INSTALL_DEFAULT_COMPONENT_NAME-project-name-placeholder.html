CMAKE_INSTALL_DEFAULT_COMPONENT_NAME-project-name-placeholder¶

  • CMAKE_INSTALL_DEFAULT_COMPONENT_NAME gained support for a special <PROJECT_NAME> placeholder value. It tells each install() invocation to take a default component name from the current PROJECT_NAME variable value.
