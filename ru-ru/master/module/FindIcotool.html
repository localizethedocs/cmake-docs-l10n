FindIcotool¶

Finds icotool, command-line program for converting and creating Win32 icon and cursor files:

find_package(Icotool [<version>] [...])

Result Variables¶

This module defines the following variables:

Icotool_FOUND

Добавлено в версии 3.3.

Boolean indicating whether (the requested version of) icotool was found.

Icotool_VERSION

Добавлено в версии 4.2.

The version of icotool found.

Cache Variables¶

The following cache variables may also be set:

ICOTOOL_EXECUTABLE

The full path to the icotool tool.

Deprecated Variables¶

The following variables are provided for backward compatibility:

ICOTOOL_FOUND

Устарело, начиная с версии 4.2: Use Icotool_FOUND, which has the same value.

Boolean indicating whether (the requested version of) icotool was found.

ICOTOOL_VERSION_STRING

Устарело, начиная с версии 4.2: Use Icotool_VERSION, which has the same value.

The version of icotool found.

Examples¶

Finding icotool and executing it in a process to create .ico icon from the source .png image located in the current source directory:

find_package(Icotool)
if(Icotool_FOUND)
  execute_process(
    COMMAND
      ${ICOTOOL_EXECUTABLE} -c -o ${CMAKE_CURRENT_BINARY_DIR}/img.ico img.png
  )
endif()
