cli-tar-lzma¶

  • The cmake(1) -E tar tool supports support LZMA compression via --lzma option.

  • The file(ARCHIVE_CREATE) command's COMPRESSION option, supports LZMA and LZMA2 compression. The LZMA2 compression is an alias for XZ.

  • The CTestCoverageCollectGCOV supports LZMA and LZMA2 compression. The LZMA2 compression is an alias for XZ.

  • The CTestCoverageCollectGCOV supports LZMA and ZSTD compression file extensions for FROM_EXT compression mode.
