ctest-script-preset-arg¶

  • The ctest_configure(), ctest_build(), ctest_test(), and ctest_memcheck() commands gained PRESET and PRESETS_FILE arguments to support using presets for their Dashboard Client steps.

  • The CTEST_CONFIGURE_PRESET, CTEST_BUILD_PRESET, CTEST_TEST_PRESET, CTEST_PRESET, and CTEST_PRESETS_FILE variables may be used to specify preset arguments to the above commands in a Dashboard Client script or on the ctest command line via the -D option.
