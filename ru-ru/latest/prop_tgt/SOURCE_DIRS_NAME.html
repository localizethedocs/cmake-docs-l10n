SOURCE_DIRS_<NAME>¶

Добавлено в версии 4.4.

Semicolon-separated list of base directories of the target's <NAME> source set, which has the set type SOURCES. The property supports generator expressions.

This property is normally only set by target_sources(FILE_SET) rather than being manipulated directly.

See SOURCE_DIRS for the list of base directories in the default source set. See SOURCE_SETS for the file set names of all source sets.
